电力行业在线考试FAQ

WHAT：电力考核在线系统考核是什么？
：在线考核是一系列有各个类型题目自动组卷为一套试卷，分配给考生来进行考核。某一个特定时间段的每一位考生的试题大都是不一样的，要么是顺序不一样，要么是试题有差别。这样能保证无舞弊行为
WHY：为什么选择电力考核在线系统考核？
：由于以往的线下考核既占用公共资源，同时效率又不高。用线上考核代替线下考核，不仅解决了有时候考核场地的不足，而且还提高了考试规划效率。
HOW：线上考核流程是怎样的？
：工作人员进行试题的录入，以及考生的录入，系统进行统一规划，统一规划时间，进行线上考核
